# WP Starter Theme

A clean starter WordPress theme built with Tailwind CSS CDN and Alpine.js CDN.

## Features

- WordPress theme standards: `wp_head`, `wp_body_open`, `wp_footer`
- Responsive header and mobile menu
- Primary and footer menu locations
- Custom logo support
- Featured image support
- Basic templates: index, page, single, archive, search, 404
- Tailwind utility classes for fast layout building
- Custom CSS layer in `assets/css/main.css`
- Alpine.js for lightweight interactions

## Install

1. Upload the ZIP file to `Appearance > Themes > Add New > Upload Theme`.
2. Activate the theme.
3. Create menus and assign them to Primary/Footer.
4. Rename theme details in `style.css` when ready.

## Recommended development path

For production, replace CDN Tailwind with a compiled Tailwind/Vite setup later. This first version is intentionally simple so you can develop fast.
